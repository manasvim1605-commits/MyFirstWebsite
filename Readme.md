## ğŸ“˜ Theory Concepts Learned

### **1. What is HTML?**
HTML (HyperText Markup Language) is the standard language used to create web pages.  
It defines the **structure** of a web page using **tags** and **elements**.

### **2. HTML Tags & Elements**
- Headings: `<h1> to <h6>`
- Paragraphs: `<p>`
- Links: `<a>`
- Images: `<img>`
- Line breaks, lists, media, etc.

### **3. HTML Document Structure**
Every HTML page contains:
- `<!DOCTYPE html>`
- `<html>`
- `<head>`
- `<body>`

### **4. Forms in HTML**
Creating user input fields using:
- `<input>`
- `<textarea>`
- `<button>`
- Attributes like `name`, `email`, `required`, `placeholder`

### **5. HTML Attributes**
Common attributes:
- `id`
- `class`
- `href`
- `src`
- `alt`

### **6. Semantic HTML**
Improves structure & readability:
- `<header>`
- `<nav>`
- `<main>`
- `<section>`
- `<footer>`
